# INSTAGRAM CLONE

## Description

This project is a simple clone of the instagram feed with infinite scroll pagination, using react native.

#### Technologies

- JavaScript
- React-Native
- Styled Components

## How To Use

#### Installation && Run

```html
    yarn install
```
```html
    yarn react-native run-ios
    or
    yarn react-native run-android
```
